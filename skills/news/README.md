# News

Latest headlines from BBC, NPR, and Reuters (RSS). No API key.

## Installation

No setup. Enable the skill in your gateway config. No env vars.

## Quick usage

- **"What's the latest news?"** → `news_headlines({ limit: 10 })`
- **"Give me 5 headlines from BBC"** → `news_headlines({ limit: 5, source: "bbc" })`

See [SKILL.md](./SKILL.md) for details.
