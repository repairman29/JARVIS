## Tools

| Tool | Description |
|------|-------------|
| `key_messages` | Key messages / talking points for a topic and audience |
| `press_release_outline` | Press release structure (headline, sections) |
| `social_post_templates` | Social post options (Twitter, LinkedIn, generic) |
| `media_pitch_outline` | Media pitch (subject line, hook, body outline) |
| `comms_brief` | Comms playbook (do's/don'ts, messages, channels) |

No env required. Tools return structured outlines; the agent uses them in conversation or expands into full copy.
