# Weather

Current weather for any city or place. No API key. Uses Open-Meteo.

## Installation

No setup. Enable the skill in your gateway config. No env vars.

## Quick usage

- **"What's the weather in Denver?"** → `weather_current({ location: "Denver" })`
- **"Temperature in London"** → `weather_current({ location: "London", units: "celsius" })`
