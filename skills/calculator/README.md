# JARVIS Calculator Skill

Advanced mathematical computing system that far surpasses Raycast's basic calculator with comprehensive calculation capabilities, real-time data integration, and intelligent context understanding.

## Installation

```bash
cp -r skills/calculator ~/jarvis/skills/
```

Add to `~/jarvis/TOOLS.md`:

```markdown
## Calculator / Mathematical Computing

**Skill:** `calculator` (installed). Advanced mathematics with units, currency, programming, and scientific calculations.

| Tool | When to use |
|------|-------------|
| `calculate` | "calculate 15% of 240", "sqrt(144)", "sin(45 degrees)" |
| `convert_units` | "convert 5 miles to km", "100 fahrenheit to celsius" |
| `convert_currency` | "convert 100 USD to EUR", "exchange rate GBP to JPY" |
| `programming_calc` | "convert FF hex to binary", "bitwise AND 1010 and 1100" |
| `financial_calc` | "compound interest 1000 at 5% for 10 years" |
| `statistics_calc` | "mean and standard deviation of [1,2,3,4,5]" |
```

This skill provides the most comprehensive mathematical computing capabilities available, making JARVIS superior to any calculator tool on the market.
