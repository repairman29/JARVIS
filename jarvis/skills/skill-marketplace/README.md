# JARVIS Skill Marketplace

Community ecosystem and extension management system that creates a thriving marketplace for JARVIS skills. This enables infinite extensibility through community contributions while maintaining security and quality standards.

## Installation

```bash
cp -r skills/skill-marketplace ~/jarvis/skills/
```

Add to `~/jarvis/TOOLS.md`:

```markdown
## Skill Marketplace / Community

**Skill:** `skill-marketplace` (installed). Community skill ecosystem and extension management.

| Tool | When to use |
|------|-------------|
| `discover_skills` | "find productivity skills", "search for Spotify integration" |
| `install_skill` | "install the GitHub skill", "add weather skill" |
| `manage_skills` | "list my skills", "update all skills", "disable music skill" |
| `create_skill_package` | "package my custom skill", "prepare skill for sharing" |
| `publish_skill` | "publish my skill to marketplace", "share with community" |
| `skill_analytics` | "what skills are trending?", "skill usage recommendations" |
```

This skill creates the community ecosystem that makes JARVIS infinitely extensible and builds a thriving marketplace of productivity tools, integrations, and automations.
