# JARVIS Workflow Automation Skill

The intelligence layer that orchestrates all JARVIS skills into powerful, reusable workflows. This is what makes JARVIS truly smart - going far beyond simple command launchers to provide AI-powered task automation that learns and adapts.

## Installation

```bash
cp -r skills/workflow-automation ~/jarvis/skills/
```

Add to `~/jarvis/TOOLS.md`:

```markdown
## Workflow Automation / AI Orchestration

**Skill:** `workflow-automation` (installed). AI-powered workflow creation and command chaining.

| Tool | When to use |
|------|-------------|
| `create_workflow` | "create morning routine workflow", "automate project setup" |
| `execute_workflow` | "run my morning routine", "execute project setup workflow" |
| `chain_commands` | "find React files, open in VS Code, snap left" |
| `workflow_templates` | "show productivity templates", "install focus mode template" |
| `ai_suggestions` | "suggest workflows for my daily routine", "optimize my workflow" |
| `learn_patterns` | "analyze my usage patterns", "suggest automation opportunities" |
```

This skill is the "brain" of JARVIS, making it the most intelligent automation system available by connecting all other skills with AI-powered orchestration.
